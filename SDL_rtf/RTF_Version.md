# RTF_Version

This function gets the version of the dynamically linked SDL_rtf library.

## Header File

Defined in SDL_rtf.h

## Syntax

```c
int RTF_Version(void);

```

## Return Value

Returns SDL_rtf version

## Version

This function is available since SDL_rtf 3.0.0.
