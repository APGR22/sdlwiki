# SDL_TTF_VERSION

This is the version number macro for the current SDL_ttf version.

## Header File

Defined in SDL_ttf.h

## Syntax

```c
#define SDL_TTF_VERSION \
    SDL_VERSIONNUM(SDL_TTF_MAJOR_VERSION, SDL_TTF_MINOR_VERSION, SDL_TTF_MICRO_VERSION)
```
