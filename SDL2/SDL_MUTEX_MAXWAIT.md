# SDL_MUTEX_MAXWAIT

This is the timeout value which corresponds to never time out.

## Header File

Defined in [SDL_mutex.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_mutex.h)

## Syntax

```c
#define SDL_MUTEX_MAXWAIT   (~(Uint32)0)
```
