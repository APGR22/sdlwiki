# SDL_hid_device_

A handle representing an open HID device

## Header File

Defined in [SDL_hidapi.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_hidapi.h), but apps should _only_ `#include "SDL.h"`!

## Syntax

```c
struct SDL_hid_device_;
```
