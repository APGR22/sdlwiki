# ASSERT_FAIL

Fails the assert.

## Header File

Defined in [SDL_test_assert.h](https://github.com/libsdl-org/SDL/blob/SDL2/include/SDL_test_assert.h), but apps should _only_ `#include "SDL.h"`!

## Syntax

```c
#define ASSERT_FAIL     0
```
