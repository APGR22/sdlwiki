= SDL_GameControllerNumMappings =

Get the number of mappings installed.

== Syntax ==

<syntaxhighlight lang='c'>
int SDL_GameControllerNumMappings(void);
</syntaxhighlight>

== Return Value ==

Returns the number of mappings.

== Version ==

This function is available since SDL 2.0.6.
