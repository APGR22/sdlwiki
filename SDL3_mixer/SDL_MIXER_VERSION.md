# SDL_MIXER_VERSION

This is the version number macro for the current SDL_mixer version.

## Header File

Defined in SDL_mixer.h

## Syntax

```c
#define SDL_MIXER_VERSION \
    SDL_VERSIONNUM(SDL_MIXER_MAJOR_VERSION, SDL_MIXER_MINOR_VERSION, SDL_MIXER_MICRO_VERSION)
```
