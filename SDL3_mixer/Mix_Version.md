# Mix_Version

This function gets the version of the dynamically linked SDL_mixer library.

## Header File

Defined in SDL_mixer.h

## Syntax

```c
int Mix_Version(void);

```

## Return Value

Returns SDL_mixer version

## Version

This function is available since SDL_mixer 3.0.0.
