# SDLNet_Version

This function gets the version of the dynamically linked SDL_net library.

## Header File

Defined in SDL_net.h

## Syntax

```c
int SDLNet_Version(void);

```

## Return Value

Returns SDL_net version

## Version

This function is available since SDL_net 3.0.0.
