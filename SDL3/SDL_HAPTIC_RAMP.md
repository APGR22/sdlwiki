# SDL_HAPTIC_RAMP

Ramp effect supported.

## Header File

Defined in [SDL_haptic.h](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_haptic.h), but apps should _only_ `#include <SDL3/SDL.h>`!

## Syntax

```c
#define SDL_HAPTIC_RAMP         (1u<<6)
```

## Remarks

Ramp haptic effect.
