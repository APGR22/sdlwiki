# SDL_SINT64_C

Append the 64 bit integer suffix to an integer literal.

## Header File

Defined in [<SDL3/SDL_stdinc.h>](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_stdinc.h)

## Syntax

```c
#define SDL_SINT64_C(c)  INT64_C(c)
```
