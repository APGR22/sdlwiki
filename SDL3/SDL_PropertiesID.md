# SDL_PropertiesID

SDL properties ID

## Header File

Defined in [SDL_properties.h](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_properties.h), but apps should _only_ `#include <SDL3/SDL.h>`!

## Syntax

```c
typedef Uint32 SDL_PropertiesID;
```
