# SDL_MS_PER_SECOND

SDL time constants

## Header File

Defined in [SDL_timer.h](https://github.com/libsdl-org/SDL/blob/main/include/SDL3/SDL_timer.h), but apps should _only_ `#include <SDL3/SDL.h>`!

## Syntax

```c
#define SDL_MS_PER_SECOND   1000
```
