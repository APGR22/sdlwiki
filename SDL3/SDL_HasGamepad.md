# SDL_HasGamepad

Return whether a gamepad is currently connected.

## Syntax

```c
SDL_bool SDL_HasGamepad(void);

```

## Return Value

Returns [SDL_TRUE](SDL_TRUE) if a gamepad is connected,
[SDL_FALSE](SDL_FALSE) otherwise.

## Version

This function is available since SDL 3.0.0.
